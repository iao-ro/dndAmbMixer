About Page

This application has been developed by Ruraidh Oliphant, Julia Pawlik and Benjamin Viney.

Our inspiration was to create an application which can be used to create different soundscapes for roleplaying games such as D&D, to make the game more immersive.
We wanted the application to be able to be used by game masters to mix together different samples to make their worlds come to life. The basis of the application is a mixer that is preloaded with audio samples.
There is volume control on each channel as well as a solo and mute button for each channel. There is an EQ for each channel and all channels can be sent to a compressor, delay and reverb effect.


How to use the mixer.

  1. From the top of the mixer select the audio source
  2. If eq is desired select from next menu, frequency and q can be adjusted by the sliders
  3. The output can then be selected: None, Master, Fx
  4. To adjust the pan use the pan knob below the output
  5. Finally the gain of the channel can be adjusted using the slider next to the meters

How to use Fx section

  1. If a channel has been sent to the Fx section just select an fx from the dropdown
  2. Controls for the fx selected should then appear
  3. Adjust to preference
  4. A second fx can be selected from the next drop down or ignored

How to use master section

  1. The play channel buttens can be used to fade sounds in and out
  2. Next to the master meter there is a master fader
